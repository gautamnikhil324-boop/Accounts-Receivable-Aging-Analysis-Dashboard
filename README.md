# Accounts-Receivable-Aging-Analysis-Dashboard
Prepared AR aging report for 300+ invoices using standard aging buckets (0-30, 31-90, 91-180, 181-360, 360+ days) .Analyzed overdue balances by salesperson, region, and customer tier to flag collection risk Found ~39% of receivables past due; summarized in a dashboard for credit control decisions Built with Excel PivotTables, slicers, and charts
